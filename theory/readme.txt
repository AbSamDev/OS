
warning! use this project only in virtual machine 

1- First download all files and paste at desktop note that makefile is without extension 
2- Run command "make" by this command multiple commands will bi created 
3- Run "sudo insmod simple.ko"
4- Check your module by "lsmod", your module may be at the top
5- Check your message that is printed by "dmesg", message maybe at the bottom
6- Removing module "sudo rmmod simple"
7- Again check removing message that is printed by "dmesg"
